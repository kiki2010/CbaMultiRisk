# CbaMultiRisk
An app focused on the province of Córdoba to find out the risk of fire and floods in your area


# Política de Privacidad

## Última actualización: 21/01/2026

Esta aplicación, MultiRisk, respeta la privacidad de sus usuarios.

## Información que se recopila

La aplicación no recopila datos personales identificables como nombre, correo electrónico o número de teléfono.

La app puede acceder a:

Ubicación aproximada (si el usuario lo permite), utilizada únicamente para mostrar información climática relevante.

Datos técnicos básicos del dispositivo necesarios para el correcto funcionamiento de la aplicación.

## Uso de la información

La información obtenida se utiliza exclusivamente para:

- Proporcionar datos climáticos actualizados

- Mejorar la experiencia del usuario

No se utilizan los datos para publicidad personalizada ni se comparten con terceros con fines comerciales.

## Contenido para niños

La aplicación está diseñada para ser apta para todo público, incluidos niños.
No contiene:

- Publicidad personalizada

- Contenido inapropiado

- Enlaces a redes sociales

- Funciones de chat o interacción entre usuarios

## Servicios de terceros

La aplicación utiliza APIs climáticas de terceros únicamente para obtener datos meteorológicos. Estos servicios no reciben información personal identificable del usuario.

## Cambios en esta política

Esta política puede actualizarse ocasionalmente. Los cambios se publicarán en esta misma página.
